Both lexica have already been included in the folder

Run the python file "quotation_detector_v3.py" to generate normaltext and quotes counter, as well as subjectivity score files.
This process takes up to 2 full days on a standalone computer.

Run the python file "subjectivity_calculator.py" to generate plots and compute statistical measures of the subjectivity lexicon.
This process only takes several seconds to complete.
